# H&M Sustainability & ESG Analysis (Python)

## Project Overview
This project analyses the H&M Group Annual & Sustainability Report 2024 using Python.
The goal is to extract, clean, and visualize ESG metrics such as carbon emissions,
energy consumption, and sustainability KPIs.

## Tools Used
- Python
- Jupyter Notebook
- Pandas
- Camelot (PDF table extraction)
- Matplotlib / Plotly

## Key Analysis
- Carbon emissions (Scope 1, 2, 3)
- ESG KPI trends
- Data extraction from sustainability reports

## Output
- Structured ESG datasets (Excel)
- Visual charts
- Reproducible analysis notebook
